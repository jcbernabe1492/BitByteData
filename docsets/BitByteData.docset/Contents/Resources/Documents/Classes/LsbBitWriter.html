LsbBitWriter

public final class LsbBitWriter : BitWriter

  • write(number:bitsCount:)

    Writes number, using and advancing by bitsCount BIT positions.

    Note

    If bitsCount is smaller than the actual amount of number‘s bits than the number will be truncated to fit into bitsCount amount of bits.

    Note

    Bits of number are processed using the same bit-numbering scheme as of the writer (i.e. LSB 0).

  • write(unsignedNumber:bitsCount:)

    Writes number, using and advancing by bitsCount BIT positions.

    Note

    If bitsCount is smaller than the actual amount of number‘s bits than the number will be truncated to fit into bitsCount amount of bits.

    Note

    Bits of number are processed using the same bit-numbering scheme as of the writer (i.e. LSB 0).

    Note

    This method is especially useful when it is necessary to write an unsigned number which overflows and, thus, crashes when converting to an Int if write(number:bitsCount:) method is used.
